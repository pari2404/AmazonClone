Amazon Clone 🛒

A responsive Amazon Clone built using HTML and CSS.
This project recreates the front-end design of the Amazon website including the navbar, hero section, product cards, and footer.

🚀 Features
🛍 Amazon-inspired homepage UI
🔍 Search bar design
📦 Product sections and cards
📱 Responsive layout
🎨 Modern styling with CSS
🧭 Navigation bar and footer
🛠 Technologies Used
HTML5
CSS3
